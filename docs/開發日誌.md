### 技術架構

#### 後端 (ASP.NET Core 8.0)
- **資料庫**：PostgreSQL
- **ORM**：Entity Framework Core
- **認證**：JWT Bearer Token
- **日誌**：Serilog
- **監控**：Prometheus metrics

#### 前端 (React 18 + TypeScript)
- **UI 框架**：Ant Design
- **狀態管理**：Redux Toolkit
- **路由**：React Router
- **HTTP 客戶端**：Axios
- **圖表**：AntV G2

#### API 端點
- `GET /api/v1/menu/latest-version` - 取得最新版本號
- `GET /api/v1/menu` - 取得菜單
- `POST /api/v1/menu` - 建立菜單
- `POST /api/v1/auth/user-login` - 使用者登入
